# 深入理解 Hugo Shortcodes

Hugo Shortcodes 是一种强大的功能，允许你在 Markdown 中嵌入复杂的 HTML 结构。

## 什么是 Shortcodes

Shortcodes 是预定义的模板片段，可以在你的内容文件中轻松调用。

### 内置 Shortcodes

Hugo 提供了许多内置的 shortcodes：

- `figure` - 插入图片
- `gist` - 嵌入 GitHub Gist
- `highlight` - 代码高亮
- `youtube` - 嵌入 YouTube 视频

### 示例代码

```go
// 这是一个 Go 代码示例
package main

import "fmt"

func main() {
    fmt.Println("Hello, Hugo!")
}
```

## 自定义 Shortcodes

你可以创建自己的 shortcodes 来满足特定需求。

### 创建步骤

1. 在 `layouts/shortcodes/` 目录下创建 HTML 文件
2. 编写模板代码
3. 在 Markdown 中调用

## 最佳实践

- 保持 shortcodes 简单明了
- 为 shortcodes 编写文档
- 测试不同的使用场景

## 结论

掌握 Shortcodes 可以大大提升你的 Hugo 使用体验。
