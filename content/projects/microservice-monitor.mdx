# 微服务监控平台

一个功能完善的分布式系统监控解决方案。

## 项目简介

本平台为微服务架构提供全方位的监控能力，包括：

- 📊 实时性能指标监控
- 📝 日志聚合与分析
- 🔔 智能告警系统
- 📈 可视化仪表板

## 架构设计

### 核心组件

1. **数据采集器**: 基于 Prometheus
2. **日志处理**: ELK Stack
3. **告警引擎**: AlertManager
4. **可视化**: Grafana

### 系统架构图

```
[微服务] --> [采集器] --> [时序数据库]
                |
                v
          [告警引擎] --> [通知服务]
                |
                v
          [可视化面板]
```

## 技术实现

### 指标采集

```go
package main

import (
    "github.com/prometheus/client_golang/prometheus"
    "github.com/prometheus/client_golang/prometheus/promhttp"
)

var (
    requestCounter = prometheus.NewCounterVec(
        prometheus.CounterOpts{
            Name: "http_requests_total",
            Help: "Total HTTP requests",
        },
        []string{"method", "endpoint"},
    )
)

func init() {
    prometheus.MustRegister(requestCounter)
}
```

### Kubernetes 集成

支持自动发现和监控 K8s 集群中的服务：

```yaml
apiVersion: v1
kind: ServiceMonitor
metadata:
  name: app-monitor
spec:
  selector:
    matchLabels:
      app: myapp
  endpoints:
  - port: metrics
```

## 部署指南

### 使用 Helm 安装

```bash
helm install monitoring ./charts/monitoring
```

### 配置告警规则

```yaml
groups:
  - name: app-alerts
    rules:
      - alert: HighErrorRate
        expr: rate(errors_total[5m]) > 0.05
        annotations:
          summary: "高错误率告警"
```
